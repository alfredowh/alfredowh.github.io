Beer Brand Classification using Deep Learning

Deep Learning Challenge, OTH Regensburg · Winter Semester 2022/2023

  • Image Classification
  • Deep Learning
  • Data Collection

Overview

As part of a Deep Learning Challenge at OTH Regensburg, our team developed a computer vision system to classify beer bottle images into 49 German brewery categories, improving upon a baseline model with approximately 80% accuracy. I contributed to designing, training, and evaluating six model variants using transfer learning, comparing architectures including EfficientNetV2, InceptionV3, Inception-ResNetV2, ResNet-18, and an Xception + ResNet-50V2 ensemble. The training strategy combined feature extraction using frozen pre-trained backbones with selective layer fine-tuning, supported by data augmentation to address the limited dataset size. The best-performing model, based on EfficientNetV2M, achieved 99.79% accuracy on the final test set.

Notes

  • Data collection: the dataset was built by photographing beer bottles across varied backgrounds, lighting conditions, perspectives, and beer types (e.g. pils, wheat beer) to increase visual variance per class.
  • Training approach: Feature extraction with a frozen pre-trained base and a new classifier head and fine-tuning of selected layers with a lower learning rate. Data augmentation (random rotation, translation, zoom, horizontal flip/crop) helped compensate for the relatively small dataset.
  • Architecture comparison: EfficientNetV2-based models outperformed Inception-ResNetV2, ResNet-18, and InceptionV3, consistent with EfficientNetV2's accuracy/efficiency trade-off from compound scaling.
