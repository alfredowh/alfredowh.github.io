Generation of Simulation Datasets for Object Detection Evaluation

M.Sc. Artificial Intelligence, TH Ingolstadt · Course Project

  • ROS2
  • Gazebo
  • Simulation
  • Data Annotation

Overview

Built a simulation object detection dataset using ROS 2 and Gazebo by simulating a simple traffic environment under diverse environmental conditions. Simulated different times of day (daylight, cloudy, sunset, and night) in Gazebo, while generating weather effects (clear, rain, and fog) using OpenCV image processing. Applied ROS 2 to manage the simulation workflow and data collection. Images are annotated in a semi-automated manner using Roboflow, which provides built-in support for automated annotation with Grounding DINO, manual correction of annotations, and exporting labels in various formats. Used this annotation tool to annotate over 3,000 objects across the Car, Pedestrian, and Truck classes. The resulting dataset was used to evaluate the robustness and performance of object detection models under varying lighting and weather conditions.
